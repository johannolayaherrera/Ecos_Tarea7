Instrucciones para obtener repositorio desde GitHub donde se encuentran los artefactos (código, formas, evidencias, etc.) :
1. En el directorio en que desea descargar el proyecto ejecute el comando : git clone https://github.com/johannolayaherrera/Ecos_Tarea7.git
2. Ingrese a la ruta ECOS - Tarea 7\src y ejecute el comando : mvn clean install

Para revisar la documentación
1. Ingrese a la carpeta \ECOS - Tarea 7\pspform\apidocs y seleccione index
